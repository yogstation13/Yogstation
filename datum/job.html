mind_traitsTraits assigned from jobs

minimal_player_ageIf you have the use_age_restriction_for_jobs config option enabled and the database set up, this option will add a requirement for players to be at least minimal_player_age days old. (meaning they first signed in at least that many days before.)

smells_likeThe text a person using olfaction will see for the job of the target's scent
space_law_notifyIf this is set to 1, a text is printed to the player when jobs are assigned, telling them that space law has been updated.
spawn_positionsHow many can start the round as this job
